ShopCAM

The dedicated advanced workflow app from the makers of ShopCalc. ShopCalc's in-app g-code surface is now ShopCAM Lite - single-op math you use ten times a day. ShopCAM is the bigger brother: chain operations, save jobs, manage machine profiles, post out to your controller. Same shop-floor DNA, bigger toolbox.

ShopCAM
What it does

Built around the problem, not around a roadmap.

The product solves something specific and gets out of your way.

  • 01

    Multi-op programs in one file

    Chain canned cycles - drill, then peck, then tap, then bore, then chamfer - across a single setup. Output is one continuous .NC program that runs end-to-end, no manual stitching at the controller.

  • 02

    Machine profiles + post-processors

    Your Haas isn't your Mazak isn't your friend's Centroid. Save per-machine post-processor settings (G-code dialect, M-code map, decimal precision, modal vs. absolute, max RPM/feed clamps) and target a specific machine on every export.

  • 03

    Saved jobs, ready to re-run

    Today's setup becomes next month's recall. Save a complete program - operations, tools, offsets, post-processor target - and re-run it when the same job lands on the table again. No re-keying.

  • 04

    Reusable templates

    Family of parts? Build the template once, plug in the dimensions per part. Hole patterns, threading recipes, fixture setups - parameterize what's variable, lock what isn't.

  • 05

    Shared tool memory with ShopCalc

    The tools you've already saved in ShopCalc's Tool Library show up in ShopCAM - same diameters, same chip loads, same SFM presets. Type once, use everywhere. No double-entry, no drift.

  • 06

    Conversational op flow

    Plain-English prompts that match how a machinist actually thinks about a setup: "What are you cutting?", "What tool are you using?", "How deep?". The post-processor handles the syntax; you handle the part.

  • 07

    Export to clipboard, file, or share

    Send the program to a .NC file, copy to clipboard for paste-into-the-controller workflows, or share via the system share sheet. No cloud dependency, no account required to export.

  • 08

    Same shop-floor feel as ShopCalc

    Dark UI that cuts shop-light glare. 64px minimum tap targets for gloves. DRO-style amber readouts. Runs offline, no telemetry. Imperial + metric in one tap. ShopCalc users will feel at home on day one.

Spec sheet

The numbers.

Everything you’d want to know about ShopCAM on a single sheet. No marketing math.

Status
In active development - no public release yet
Platforms
iOS + Android (mobile-first, same as ShopCalc family)
Pricing model
Premium one-time + subscription tier. Pricing finalized at launch.
What lives in ShopCalc instead
Single-operation generators are ShopCAM Lite, built into ShopCalc Pro: G81/82/83/84/85 canned cycles + helical threadmilling. See ShopCalc.
Sister apps
Same makers as ShopCalc, WelderCalc, BuilderCalc, ElectricianCalc - all shop-floor offline calculators on the App Store + Google Play.
Account required
No - tool memory syncs locally via the operating-system app group; cloud sync is optional and opt-in.
Offline
Yes. Same offline-first design as ShopCalc. No network required to generate code.
